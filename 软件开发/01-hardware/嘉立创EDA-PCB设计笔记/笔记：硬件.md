# 阻抗匹配基本原理



记住下面这个描述：

频率(波形每秒抖动多少次)
波长(波形抖动一次，波往前传多远)

**高速信号必须考虑阻抗匹配**

**阻抗匹配**，就是从信号源到负载，这整条链路中的阻抗不能有突变(突然变大or突然变小都不行)，否则突变点必然发生反射or部分反射，反射回的波形会干扰正在传输的波形，从而导致信号混叠失真。

阻抗不怕变，只怕 “突然变”。

---


**为什么 50Ω 是通用阻抗？为什么差分线是 100Ω？**

50Ω 不是上帝规定的，是同轴电缆 + PCB 损耗 + 功率承受能力三者打架，最后妥协出来的 “黄金中间值”。

**为什么同轴电缆最后选了 50Ω？**

早年射频都是**同轴电缆**，决定阻抗的就 3 个指标：传输损耗（越小越好）、耐压 / 功率容量（越大越好）、物理尺寸、工艺方便性。工程师一算发现：损耗最小的阻抗 ≈ 77Ω，功率容量最大的阻抗 ≈ 30Ω，77 和 30，取个**中间好用的整数 50Ω**，大家一统一，**50Ω 就成了世界通用标准**。后来 PCB 射频、高速单端信号，直接沿用：**50Ω 单端**。

差分阻抗定义：差分阻抗 = 正 + 负，Zdiff = Z0 + Z0 ≈ 100Ω

**工程上为什么大家都死磕这两个值？**

因为 芯片引脚、封装、库文件、仿真模型、驱动能力全是按 **50/100Ω** 设计的。PCB 厂家**只对 50/100Ω 最熟练、最稳、最便宜**。连接器、电缆、测试设备，全是按照 50/100Ω 行业标准生产的。你不按这个来：不好做不好测不好匹配容易信号崩。


<font color=blue>示波器测量信号的探头，为什么默认是1MΩ的高阻抗？
</font>

示波器1MΩ高阻抗信号探头，相当于一根超级细、阻力巨大的细管子，水流很难流过去，几乎不 “抢水”。（根据并联分流原理）当示波器探头测量电路的波形时，也就不会把电路的电流吸走，否则，示波器测量电路的波形时，顺带还吸走电路的电流，那就测不准了。


<font color=blue>那为什么示波器测高频不用1MΩ阻抗？这样不吸走高频信号的电流不是更好么？</font>

先分清两个完全不同的问题：低频要：**尽量别吸电流** → 1MΩ（超细管）；高频要：**必须把波彻底吃掉、不反弹** → 50Ω（标准匹配管）。

传输线是 **50Ω**（标准粗细水管），你接到示波器 **1MΩ**（超级细管，几乎堵死），这等于波冲过来：穿不过超细管，就像撞上了一堵墙，波形信号直接全部反弹了，这就是开路反射。反射带来的灾难：你测到的是 “假波形”，你看到的根本不是真实信号，是波撞来撞去的乱码。
